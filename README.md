# Nestjs starter kit 

## Description
A comprehensive starter kit for creating efficient backend services. Developed due to frequent encounters with the tedious process of installing and setting up NestJS along with essential modules, this project aims to simplify the process by sharing preconfigured configurations. 

# Modules
- Auth module with JWT
- ConfigModule
- HealthCheck

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
